# milanSite

Статический сайт для публикации через GitHub Pages.

## Локальный запуск

```bash
python3 -m http.server 8000
```

Открыть:

`http://localhost:8000/`

## GitHub Pages

1. Создать пустой репозиторий на GitHub.
2. Подключить `origin`.
3. Запушить ветку.
4. Включить GitHub Pages из ветки `main`.
